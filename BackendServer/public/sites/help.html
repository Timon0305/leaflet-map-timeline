Abnehmer JSON

Zum korrekten import muss für jeden Abnehmer eine JSON Datei erstellt werden. Die Abnehmer JSON Dateien sowie die Heizhaus JSON müssen sich danach in einem Ordner zum import befinden. Im Abschnitt "historyData" muss für jeden Zeit Eintrag ein eigenes Array Element erstellt werden (Siehe Beispiel JSON).

Folgende eigenschaften der Abnehmer JSON müssen erfüllt sein.
  • Eindeudige id
  • Eindeudige customerId
  • Vollständige Adressdaten damit eine automatische zuweisung zu Koordinaten erfolgen kann
  • Historische Daten
  • Dateinamen können frei gewählt werden
Tipp: Die Abnehmer JSON Dateien können unter "Diverses -> Abnehmer JSON prüfen" analysiert werden.

Heizhaus JSON

Heizhaus daten sind optional. Die Abnehmer JSON Dateien sowie die Heizhaus JSON müssen sich danach in einem Ordner zum import befinden. Im Abschnitt "historyData" muss für jeden Zeit Eintrag ein eigenes Array Element erstellt werden (Siehe Beispiel JSON).

Folgende eigenschaften der Heizhaus JSON müssen erfüllt sein.
  • Eindeudige id
  • Vollständige Adressdaten damit eine automatische zuweisung zu Koordinaten erfolgen kann
  • Historische Daten
  • Dateiname können frei gewählt werden
Tipp: Die Heizhaus JSON Dateien können unter "Diverses -> JSON prüfen" analysiert werden.
